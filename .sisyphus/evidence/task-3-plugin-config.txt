## Configuration in src-tauri/src/lib.rs (lines 142-146)
        // Exclude floating windows ("floating-" prefix) from state persistence --
        // they are ephemeral and managed by FloatingWindowRegistry, not the plugin.
        .plugin(tauri_plugin_window_state::Builder::default()
            .with_filter(|label| !label.starts_with("floating-"))
            .build())

## Effect
- Main window: state IS saved/restored (label "main" does not start with "floating-")
- Floating windows: state NOT saved/restored (labels like "floating-<uuid>" are excluded)
