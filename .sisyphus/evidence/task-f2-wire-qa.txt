BLOCKER: TAURI NATIVE CONTEXT REQUIRED
---------------------------------------
The app requires window.__TAURI_INTERNALS__ APIs (getCurrentWindow, listen, transformCallback)
which are only available in the actual Tauri native WebView. When navigating to the Vite dev
server in a plain browser (Playwright), the React app fails to mount because:
  - getCurrentWindow() returns undefined (no __TAURI_INTERNALS__)
  - listen() / transformCallback() throw TypeError
  - Root div remains empty; PixiJS canvas never initializes

Visual screenshots were captured but show a blank white page (app not rendered).
Runtime testing requires `pnpm tauri dev` which was excluded from scope.
